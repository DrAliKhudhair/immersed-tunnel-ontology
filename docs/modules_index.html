Documentation Structure Overview

This ontology follows the modular architecture specified in GBT 51318-2019 with 20 specialized modules organized into three main categories:

  • 4 Core Foundation Modules: Core semantic infrastructure and taxonomic organization
  • 6 System Modules: Operational and safety systems for tunnel infrastructure
  • 10 Technical & Management Modules: Engineering domains with comprehensive coverage
