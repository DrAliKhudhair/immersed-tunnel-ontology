Immersed Tunnel Ontology - Core Foundation Module: Description back to ToC

Core foundation module establishing fundamental base classes for immersed tunnel engineering ontology. Aligned with GBT 51318-2019 "Immersed Tunnel Design Standard" providing standardized semantic infrastructure. This module defines foundational concepts including: - Calculation stages corresponding to construction phases per GBT 51318-2019 methodology - Limit states for structural analysis and safety verification - Project deliverables and documentation framework - Base component classifications for tunnel elements STANDARDS ALIGNMENT: - Construction stages follow GBT 51318-2019 design methodology - Terminology consistent with Chinese national standard definitions - Structural analysis framework supporting limit state design approach
